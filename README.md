# RandomSpawnPlus5

🔀 A random spawn plugin for modern Minecraft version

## Compatibility

- Support Java 8 and higher
- Support 1.12.2 - Latest Minecraft version (1.20.3)
- Compatible with Spigot / Paper / Paper Forks / Forge+Bukkit Hybrid Server
- Folia Support

## Building

`./gradlew clean build`
